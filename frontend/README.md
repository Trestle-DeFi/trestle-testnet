# Testnet Frontend

Next.js frontend for the Trestle DeFi testnet platform. Uses wagmi + Reown AppKit for wallet connection and contract interaction.

## Commands

```bash
npm install
npm run dev    # http://localhost:3000
npm run build
npm start
```
